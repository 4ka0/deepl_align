## DeepL Align

A command line script for translating a docx file using the DeepL API.<br><br>
The translation is output as a tmx file or as a docx file table with the source and target text segments ***fully aligned***.

### Built using:

* Python 3.10
* Django 4.1.3
* Bootstrap 5
* HTMX
* Docker
* Gunicorn
* Nginx
* PostgreSQL
